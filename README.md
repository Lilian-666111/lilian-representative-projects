project_climate: Climate Change Statistical Analysis Project

Conducted a statistical analysis of climate change patterns using historical weather data from the United States, with the goal of evaluating the significance of long-term temperature variations and extreme drought events. The project first performed exploratory analysis on the geographical distribution of weather stations and classified cities into different regions based on their geographic characteristics. Bootstrap resampling methods were then applied to construct confidence intervals for Phoenix temperature data, enabling hypothesis testing on whether recent temperature levels significantly differed from historical averages. In addition, a permutation test framework was developed to examine whether reduced precipitation during drought years in the Southwestern United States was statistically significant by generating empirical null distributions through randomization.

My primary contributions included data preprocessing, statistical modeling, implementation of Bootstrap and Permutation Test algorithms, hypothesis testing, and visualization of analytical results. By transforming real-world climate questions into statistical inference problems, I developed a complete analytical workflow covering data exploration, hypothesis formulation, random simulation, and evidence-based conclusion evaluation, strengthening my ability to apply statistical methodologies to real-world datasets.

Distributed Computing Projects:

In the Distributed Computing course projects, I worked on designing distributed optimization and machine learning systems based on the Ray framework. I implemented distributed optimization algorithms by parallelizing data processing, gradient computation, and cross-validation procedures, and evaluated the performance improvement under different parallel settings. Also, I further applied distributed computing techniques to machine learning training by designing a Ray Actor-based architecture for data parallel training, including model state management and gradient synchronization. Through these projects, I gained practical experience in distributed task scheduling, stateful computation, and scalable machine learning systems.
